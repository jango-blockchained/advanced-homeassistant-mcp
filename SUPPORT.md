If you've found a bug or issue:

1. **Search existing issues**: Check if someone has already reported it
   - [Open Issues](https://github.com/jango-blockchained/advanced-homeassistant-mcp/issues)
   - [Closed Issues](https://github.com/jango-blockchained/advanced-homeassistant-mcp/issues?q=is%3Aissue+is%3Aclosed)

2. **Create a new issue**: If not found, [open a new issue](https://github.com/jango-blockchained/advanced-homeassistant-mcp/issues/new) with:
   - Clear, descriptive title
   - Detailed description of the problem
   - Steps to reproduce
   - Expected vs actual behavior
   - Environment details (OS, Bun version, Home Assistant version)
   - Relevant logs or error messages (sanitize sensitive data)
   - Screenshots if applicable
